# github
タイトルFlaskを使用してチャットアプリを作成して、Webアプリについて学習をしました！
## 1.アプリの概要と機能
サーバーサイドにPython、フレームワークのFlask、インフラにAWS、サーバーOSにApache2、WSGIにuWSGIを使用し、GithubとCodeDeployを使用して半自動でデプロイ可能な環境を構築しました。
* 機能としては
チャット機能
ユーザー登録機能
認証機能
